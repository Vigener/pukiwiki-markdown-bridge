## 定型ページの自動補完・自動作成機能

- [ ] **日付などの自動補完・ページ自動作成機能**
  - 例：毎週新しいページを作る際に `[06/30](./20260630)` のように打つ手間を省きたい。
  - アイデア1：`[06/30]` まで打つと自動補完される。
  - アイデア2：テンプレートボタンを押すとカレンダーが出てきて、追加したい日付を選択すると、ユーザーがあらかじめ決めたフォーマットでページを作成してくれる。

### 参考：毎週の進捗のページのイメージ
```markdown
[Architecture-team/igarashi](./Architecture-team/igarashi)<!--PW_LINK-->

# やったこと
- 

# やること
- 

# スライド
#ref(ArchMTG_{YYYYMMDD}_igarashi.pdf);
```

### 汎用性に関する検討事項
- 上記のようなページを作成する際、対象のページURLなどから日付等（例: `Architecture-team/igarashi/{YYYYMMDD}`）を自動取得できれば便利だが、ディレクトリ構成はユーザーによって異なる。
- 標準機能として組み込むのではなく、**各ユーザーが柔軟にマクロを組める（変数を定義できる）ような汎用性の高い設計**を検討する必要がある。
